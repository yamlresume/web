印度尼西亚语 (Bahasa Indonesia) 是
[印度尼西亚](https://en.wikipedia.org/wiki/Indonesian_language)的官方语言，
这是世界上最大的群岛国家。它作为印度尼西亚多元人口的通用语，有超过2.7亿人
讲数百种母语。

## 语言学

### 字符集

印度尼西亚语使用[拉丁字母](https://en.wikipedia.org/wiki/Latin_alphabet)，
与英语使用相同的书写系统。它遵循标准的26个字母表，没有额外字符。

| 特性               | 英语字符集              | 印度尼西亚语字符集          |
| ------------------ | ---------------------- | ------------------------- |
| **字母表**         | 26个字母 (`a-z`)       | 26个字母 (`a-z`)          |
| **变音符号**       | 无                     | 无                        |
| **特殊字符**       | 限于基本拉丁字符        | 限于基本拉丁字符           |

#### 地点

印度尼西亚语地址遵循特定格式：

1.  **街道和门牌号**：街道名称在前，后面是门牌号（例如" Jalan Sudirman No. 123"）。
2.  **村/街道 (Kelurahan/Desa)**：最小行政单位。
3.  **区 (Kecamatan)**：下一级行政单位。
4.  **城市 (Kota/Kabupaten)**：城市或县。
5.  **省 (Provinsi)**：最大行政区划。
6.  **邮政编码**：5位数邮政编码放在最后。

#### 大写

印度尼西亚语遵循与英语类似的大写规则：

- 句子的第一个单词大写。
- 专有名词大写（人名、地名、组织名）。
- 星期和月份不大写。
- 语言名称和国籍不大写。

### 排版

#### 标点

印度尼西亚语标点规则与英语非常相似：

- **引号**：使用双引号 `"..."`（美式）。
- **撇号**：使用方式与英语相似。
- **逗号**：遵循英语规则。
- **数字**：
  - **小数分隔符**：使用逗号（例如 `12,34`）。
  - **千位分隔符**：使用点（例如 `1.234`）。
