---
title: Motor HTML
---

O motor HTML gera versões HTML limpas e prontas para a web do seu
currículo. Isto é particularmente útil para incorporar o seu currículo em sites,
partilhá-lo online ou criar um portfólio baseado na web.

## Configuração

Para utilizar o motor HTML, especifique `engine: html` na sua configuração de
layout.

```yaml
layouts:
  - engine: html
    template: calm
    # ... outras opções
```

## Secções

A opção `sections` permite personalizar os títulos das secções e a sua
ordem. Para mais informações, consulte a [documentação de
secções](./sections).

```yml lineNumbers
layouts:
  - engine: html
    sections:
      aliases:
        work: "Experiência Profissional"
        education: "Educação"
        skills: "Competências"
      order:
        - basics
        - work
        - education
```

## Modelos

A opção `template` permite escolher entre uma variedade de modelos. Cada
modelo tem um design e disposição únicos. Pode encontrar uma lista de
modelos disponíveis na [documentação de modelos](./html/templates).

```yml lineNumbers
layouts:
  - engine: html
    template: calm
```

## Tipografia

A opção `typography` permite personalizar os estilos de fonte utilizados
no seu currículo.

### Tamanho da fonte

A opção `fontSize` permite definir o tamanho de fonte base para o documento.
O motor HTML suporta tamanhos de fonte entre `14px` e `20px`.

### Família de fontes

A opção `fontFamily` permite especificar uma lista priorizada de nomes de famílias
de fontes e/ou nomes genéricos de famílias. Isto funciona exatamente como a
propriedade CSS `font-family`.

### Espaçamento de linha

A opção `lineSpacing` permite ajustar o espaçamento de linha do documento.
YAMLResume utiliza valores semânticos que são mapeados para implementações específicas do motor:

| Valor    | Descrição                                    |
| -------- | -------------------------------------------- |
| `tight`  | Espaçamento compacto para caber mais conteúdo |
| `snug`   | Um pouco mais de espaço do que tight        |
| `normal` | Legibilidade equilibrada (predefinição)     |
| `relaxed`| Mais espaço entre linhas                   |
| `loose`  | Layout muito espaçado                      |

```yaml
layouts:
  - engine: html
    typography:
      fontSize: 16px
      fontFamily: "Merriweather, serif"
      lineSpacing: relaxed
```

## Avançado

A opção `advanced` permite configurar definições avançadas para o motor HTML.

```yml lineNumbers
layouts:
  - engine: html
    advanced:
      showIcons: true
      # Título personalizado para o documento HTML.
      title: "O Meu Título de Currículo Personalizado"
      # Rodapé personalizado para o documento HTML.
      footer: "Gerado por <a href=\"https://yamlresume.dev\">YAMLResume</a>"
      # Meta descrição para o documento HTML.
      description: "Uma breve descrição do currículo para SEO."
      # Meta palavras-chave para o documento HTML.
      keywords: "currículo, cv, yamlresume"
```
