## Implementação

VS Code é um tema escuro inspirado no esquema de cores "Dark+" do Visual
Studio Code. Foi concebido especificamente para developers, engenheiros de
software e profissionais técnicos que desejam que o seu currículo reflita o seu
ambiente de codificação.

O modelo adota uma estética de "Currículo como Código", tratando diferentes
partes do seu perfil como elementos de código:

- **Nome**: Decorado como uma `class` (ex., `class JohnDoe {`)
- **Cabeçalho**: Representado como um comentário de código (ex.,
  `// Full Stack Engineer`)
- **Secções**: Representadas como chamadas de funções (ex., `experience()`)
- **Habilidades/Idiomas**: Estilizados como chaves de propriedade de objeto (ex.,
  `JavaScript: Expert`)

Por baixo do capô, o modelo utiliza:

- **Tipografia**: Utiliza uma pilha de fontes monoespaçadas (`Consolas`,
  `Monaco`, `Courier New`, `monospace`) para uma sensação de IDE autêntica
- **Realce de Sintaxe**: Utiliza a paleta de cores oficial do VS Code
  Dark+ para palavras-chave, cadeias, classes e funções
- **Sangria de Design**: Apresenta guias de sangria vertical (borda esquerda)
  para representar visualmente o alcance de cada secção

## Características

- **Centrado no Developer**: Perfeito para destacar competência técnica e
  passion pela programação
- **Modo Escuro por Defeito**: Tema escuro de alto contraste, agradável à
  vista
- **Estrutura Tipo Código**: Metáforas visuais únicas que fazem com que os
  currículos técnicos se destaquem
- **Responsivo**: Totalmente adaptado para visualização tanto em desktop
  como em dispositivos móveis com semântica de código consistente
- **Estilo de Sintaxe**: Utiliza cores familiares de IDE para diferenciar
  entre papéis, datas e descrições

## Pré-visualização

O modelo VS Code oferece um aspeto distintivo e moderno que conecta
imediatamente com recrutadores técnicos e gestores de contratação. Transforma o
seu historial profissional num "ficheiro fonte" limpo, legível e visualmente
familiar.
