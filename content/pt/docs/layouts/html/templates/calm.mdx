## Implementação

Calm é um modelo HTML limpo e moderno inspirado no modelo LaTeX
moderncv.  Apresenta um design minimalista com separação clara de secções e um
design elegante otimizado para visualização web.

Internamente, o modelo utiliza:

- **Reinício CSS**: Uma folha de estilos de reinício personalizada para
  renderização consistente entre navegadores
- **Tipografia**: Tamanhos de fonte personalizáveis que variam de 14px até
  20px
- **Design Responsivo**: Design limpo que se adapta a diferentes tamanhos de
  ecrã
- **HTML Semântico**: Estrutura HTML5 adequada para melhor acessibilidade e SEO

## Configuração

```yml lineNumbers title="resume.yml"
content:
  # ...

layouts:
  - engine: html
    template: calm    # [!code highlight]
    typography:
      fontSize: 16px
```

## Características

- **Design Limpo**: Design minimalista focado na legibilidade do conteúdo
- **Tipografia Moderna**: Fontes profissionais com tamanhos personalizáveis
- **Pronto para Web**: Pode ser incorporado diretamente em sites ou partilhado
  através de URL
- **Apropriado para Impressão**: Otimizado para imprimir em PDF a partir de navegadores web
- **Secções Personalizáveis**: Ordenação e nomenclatura flexível de secções

## Pré-visualização

O modelo Calm proporciona uma aparência profissional adequada para diversas
indústrias e níveis de experiência. O seu design limpo garante que o seu conteúdo
seja o centro de atenção enquanto mantém o apelo visual.
