## Implementação

Moderncv classic é um modelo de currículo moderno e clássico baseado em
[xdanaux/moderncv](https://github.com/xdanaux/moderncv).

Por baixo do capô, usamos o seguinte código para o preâmbulo do LaTeX:

```latex lineNumbers
\documentclass[a4paper, serif, 11pt]{moderncv}

%% moderncv
% style and color
\moderncvstyle{classic}
\moderncvcolor{black}

% needed by moderncv for showing icons
\usepackage{fontawesome5}
```
