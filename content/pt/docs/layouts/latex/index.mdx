---
title: Motor LaTeX
---

O motor LaTeX é o motor central do YAMLResume, concebido para produzir currículos em PDF de alta qualidade e aspeto profissional utilizando modelos LaTeX.

## Configuração

Para utilizar o motor LaTeX, especifique `engine: latex` na sua configuração de layout.

```yaml
layouts:
  - engine: latex
    template: moderncv-banking
    # ... outras opções
```

## Página

A opção `page` permite personalizar a configuração da página, como o
tamanho do papel, as margens e se mostrar os números de página.

```yml lineNumbers
layouts:
  - engine: latex
    page:
      showPageNumbers: true
      # a4 ou letter
      paperSize: a4
      margins:
        top: 2.5cm
        left: 1.5cm
        right: 1.5cm
        bottom: 2.5cm
```

## Secções

A opção `sections` permite personalizar os títulos das secções e a sua ordem. Para mais informações, consulte a [documentação de secções](./sections).

```yml lineNumbers
layouts:
  - engine: latex
    sections:
      aliases:
        work: "Experiência Profissional"
        education: "Educação"
        skills: "Competências"
      order:
        - basics
        - work
        - education
```

## Modelos

A opção `template` permite escolher entre uma variedade de modelos. Cada modelo tem um design e disposição únicos. Pode encontrar uma lista de modelos disponíveis na [documentação de modelos](./latex/templates).

```yml lineNumbers
layouts:
  - engine: latex
    template: moderncv-banking
```

## Tipografia

A opção `typography` permite personalizar os estilos de fonte utilizados no seu currículo.

### Tamanho da fonte

A opção `fontSize` permite definir o tamanho de fonte base para o documento.
O motor LaTeX apenas suporta `10pt`, `11pt` e `12pt`.

### Família de fontes

A opção `fontFamily` permite especificar uma lista priorizada de nomes de famílias
de fontes. Isto funciona de forma semelhante à propriedade CSS `font-family`.

```yaml
layouts:
  - engine: latex
    typography:
      fontSize: 11pt
      fontFamily: "Merriweather, serif"
```

## Avançado

A opção `advanced` permite configurar definições avançadas para o motor
LaTeX.

```yml lineNumbers
layouts:
  - engine: latex
    advanced:
      showIcons: true
      fontspec:
        numbers: Lining
```
