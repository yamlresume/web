## JSON Resume

[JSON Resume](https://jsonresume.org) é um pioneiro na construção de um
padrão aberto para um formato de dados estruturados para currículos. O seu
primeiro [lançamento oficial](https://jsonresume.org/blog/first-official-release)
foi a 31 de julho de 2014, há mais de 10 anos.

Do seu site oficial:

````markdown
## Origens e Conceptualização

JSON Resume foi concebido como uma solução ao panorama fragmentado e
inconsistente de formatos de currículos. Os currículos tradicionais,
tipicamente criados em formatos como Microsoft Word ou PDF, frequentemente sofrem de
problemas de compatibilidade quando são analisados por diferentes sistemas.
Reconhecendo este problema, Thomas Davis e Roland Sharp iniciaram JSON Resume
como um projeto de código aberto para criar um formato universal e legível por
máquina para currículos.

## Marcos de Desenvolvimento

O projeto começou a ganhar impulso no início da década de 2010, com marcos
significativos como a criação do esquema JSON Resume, o desenvolvimento das
primeiras ferramentas CLI, e o estabelecimento do registo JSON Resume. Estes
marcos foram impulsionados por contribuições da comunidade e o crescente
reconhecimento da necessidade de um formato de currículo padronizado na
indústria tecnológica. ```

JSON Resume fez um excelente trabalho na construção de um padrão aberto
para um formato de dados estruturados para currículos. No entanto, falta-lhe
algumas funcionalidades que o YAMLResume tem.

O YAMLResume é altamente inspirado por [JSON Resume](https://jsonresume.org/),
e é mais opinado em alguns aspetos. Por exemplo, JSON Resume não diz nada
sobre como converter os dados legíveis por máquina num currículo belo e
legível por humanos.

Aqui estão algumas diferenças notáveis:

| Funcionalidade          | YAMLResume               | JSON Resume              |
| ---------------------- | ------------------------ | ------------------------ |
| Formato de Dados       | YAML                     | JSON                     |
| Estrutura de Dados     | Todos objetos nível raiz | Alguns objetos aninhados |
| Motor de Composição    | LaTeX                    | Nenhum                   |
| Especificação Design  | Integrada                | Não especificada         |
| Suporte Texto Rico     | Sim                      | Não                      |

Vamos oferecer uma ferramenta para importar JSON Resume para YAMLResume no
futuro.
