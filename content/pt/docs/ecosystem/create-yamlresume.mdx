O [create-yamlresume](https://yamlresume.dev/docs/ecosystem/create-yamlresume)
ajuda-te a criar um novo projeto [YAMLResume](https://yamlresume.dev) com um
comando de uma linha.

## Utilização

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn', 'bun']}>
```console tab="npm"
npx create-yamlresume my-resume

# ou

npm init yamlresume my-resume
```

```console tab="pnpm"
pnpm create yamlresume my-resume
```

```console tab="yarn"
yarn create yamlresume my-resume
```

```console tab="bun"
bun create yamlresume my-resume
```

</Tabs>

## O que faz

Esta ferramenta irá:

1. Criar um novo diretório com o nome do teu projeto
2. Criar uma estrutura básica de projeto Node.js com:
   - `package.json` com dependência do YAMLResume e scripts úteis
   - ficheiro `.gitignore` com exclusões apropriadas
   - `README.md` com instruções básicas de utilização
3. Instalar todas as dependências necessárias
4. Criar um novo ficheiro de currículo de exemplo usando `yamlresume new`
5. Mostrar comandos disponíveis e próximos passos

## Estrutura do Projeto

O projeto gerado apresenta-se assim:

```
my-resume/
├── package.json          # Configuração do projeto com dependência yamlresume
├── .gitignore            # Regras de ignore do Git
├── README.md             # Documentação do projeto
└── resume.yml            # O teu currículo YAML (nome de ficheiro que escolheres)
```

## Comandos Disponíveis

No projeto gerado, podes executar:

- `npm run build` - Construir o teu currículo para PDF
- `npm run dev` - Vigiar alterações e reconstruir automaticamente
- `npm run validate` - Validar o teu currículo contra o esquema
- `npm run yamlresume` - Executar o CLI do YAMLResume
