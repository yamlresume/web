O YAMLResume fornece um conjunto de ferramentas para te ajudar a criar,
converter e gerir os teus currículos de forma mais eficiente. Aqui estão
algumas das utilidades principais disponíveis:

## create-yamlresume

O [create-yamlresume](./ecosystem/create-yamlresume) facilita iniciar um novo
projeto de YAMLResume com um comando de uma linha. Irá estruturar o diretório
do teu projeto, instalar as dependências necessárias e gerar um arquivo de
currículo de exemplo para que possas começar de imediato.

- Estrutura um novo projeto com `npx create-yamlresume my-resume` ou comandos
  similares para `npm`, `yarn`, ou `pnpm`.
- Inclui uma estrutura de projeto pronta a usar, scripts para construir e
  validar currículos, e um arquivo de currículo YAML de exemplo.

## json2yamlresume

O [json2yamlresume](./ecosystem/json2yamlresume) é uma ferramenta CLI para
converter arquivos de [JSON Resume](https://jsonresume.org/) para o formato
YAMLResume.

- Converte ficheiros JSON Resume para YAMLResume, transformando campos e
  estrutura conforme necessário.
- Garante que os teus dados são validados contra o esquema do YAMLResume.

Estas ferramentas ajudam-te a migrar de outros formatos, automatizar a
configuração do projeto e otimizar o teu fluxo de trabalho com YAMLResume.
