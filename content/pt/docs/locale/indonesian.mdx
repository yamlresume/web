O indonésio (Bahasa Indonesia) é o
[idioma oficial](https://en.wikipedia.org/wiki/Indonesian_language) da
Indonésia, a maior nação arquipelágica do mundo. Serve como língua franca
entre a população diversificada da Indonésia, com mais de 270 milhões de pessoas
falando centenas de idiomas nativos.

## Linguística

### Conjunto de Caracteres

O indonésio usa o [alfabeto latino](https://en.wikipedia.org/wiki/Latin_alphabet),
o mesmo sistema de escrita do inglês. Segue o alfabeto padrão de 26 letras sem
caracteres adicionais.

| Recurso                | Conjunto de Caracteres em Inglês   | Conjunto de Caracteres em Indonésio   |
| ---------------------- | --------------------------------- | ------------------------------------ |
| **Alfabeto**           | 26 letras (`a-z`)                | 26 letras (`a-z`)                   |
| **Diacríticos**        | Nenhum                            | Nenhum                               |
| **Caracteres Especiais** | Limitado a caracteres latinos básicos | Limitado a caracteres latinos básicos |

### Modismos

#### Data

As datas em indonésio seguem o formato dia-mês-ano. Os nomes dos meses não
são capitalizados.

| --------- | ---------- |
| January   | Januari    |
| February  | Pebruari   |
| March     | Maret      |
| April     | April      |
| May       | Mei        |
| June      | Juni       |
| July      | Juli       |
| August    | Agustus    |
| September | September  |

| December  | Desember   |

#### Localização

Os endereços indonésios seguem um formato específico:

1.  **Rua e Número**: O nome da rua vem primeiro, seguido do
    número da casa (exemplo: "Jalan Sudirman No. 123").
2.  **Bairro (Kelurahan/Desa)**: A menor área administrativa.
3.  **Distrito (Kecamatan)**: O próximo nível administrativo.
4.  **Cidade (Kota/Kabupaten)**: A cidade ou regência.
5.  **Província (Provinsi)**: A maior divisão administrativa.
6.  **CEP**: O código postal de 5 dígitos vem no final.

#### Capitalização

O indonésio segue regras de capitalização semelhantes ao inglês:

- Capitalize a primeira palavra de uma frase.
- Capitalize nomes próprios (nomes de pessoas, lugares, organizações).
- Dias e meses não são capitalizados.
- Nomes de idiomas e nacionalidades não são capitalizados.

### Composição Tipográfica

#### Pontuação

As regras de pontuação indonesas são muito semelhantes às do inglês:

- **Aspas**: Usa aspas duplas `"..."` (estilo americano).
- **Apóstrofo**: Usado de forma semelhante ao inglês.
- **Vírgula**: Segue as regras do inglês.
- **Números**:
  - **Separador decimal**: Usa uma vírgula (exemplo: `12,34`).
  - **Separador de milhar**: Usa um ponto (exemplo: `1.234`).
