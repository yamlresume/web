O japonês (日本語) é uma língua da Ásia Oriental falada principalmente no
Japão. É membro da família de línguas japónicas.

## Linguística

### Conjunto de caracteres

O sistema de escrita japonês utiliza uma combinação de três escritas:
Kanji (caracteres chineses adotados), Hiragana e Katakana. A escrita latina
(Romaji) também é utilizada para certos propósitos.

### Suporte CJK

O YAMLResume detecta automaticamente o japonês como um idioma CJK e aplica
ajustes de design específicos, como o uso de números de estilo "Lining" em
vez de "OldStyle" para que coincidam melhor com os glifos CJK.

## Modismos

### Data

As datas japonesas são formatadas na ordem Ano, Mês e Dia, utilizando
contadores específicos.

| Inglês    | Japonês |
| --------- | ------- |
| January   | 1月     |
| February  | 2月     |
| March     | 3月     |
| April     | 4月     |
| May       | 5月     |
| June      | 6月     |
| July      | 7月     |
| August    | 8月     |
| September | 9月     |
| October   | 10月    |
| November  | 11月    |
| December  | 12月    |

Os intervalos de datas utilizam o til de largura completa (U+FF5E) como separador
(por exemplo, 2016年10月～2018年1月).

### Localização

No currículo gerado, os endereços japoneses geralmente seguem a
ordem de País, Região (Prefectura), Cidade, Endereço e finalmente Código
Postal.

Nota: Embora os endereços japoneses padrão geralmente comecem com o
Código Postal (〒), o YAMLResume atualmente adiciona-o no final para manter a
coerência com as variáveis de estrutura geradas.

## Composição tipográfica

### Pontuação

O japonês utiliza sinais de pontuação de largura completa.

- **Vírgula**: `、` (Vírgula ideográfica, U+3001) é utilizada em vez da
  vírgula latina `,`.
- **Dois pontos**: `：` (Dois pontos de largura completa, U+FF1A) é utilizado
  em vez dos dois pontos latinos `:`.
- **Separador**: `、` também é utilizado como separador de listas.
