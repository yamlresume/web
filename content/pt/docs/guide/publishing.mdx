## Formato de arquivo

Os formatos de arquivo comuns para currículos incluem:

- PDF
- Microsoft Word (doc, docx)
- Texto Simples
- HTML

Aqui recomendamos mais o PDF:

- PDF tem excelente qualidade de saída multiplataforma, assegurando impressão e
  visualização quase 100% consistentes
- PDF pode ser incorporando com fontes, permitindo grande liberdade na
  seleção de fontes
- leitor PDF padrão disponível na maioria das plataformas
  - Windows 8 e superior têm um leitor PDF integrado
  - macOS tem
    [Preview](<https://en.wikipedia.org/wiki/Preview_(Mac_OS)>), iOS
    tem [iBooks](http://www.apple.com/ibooks/)
  - muitos navegadores (como Chrome) têm
    [pdf.js](http://mozilla.github.io/pdf.js/) integrado, que também pode abrir
    PDF por padrão

É mais fácil aprender MS Word, mas entregar um currículo com formato Word
tem grandes desvantagens:

- Os documentos de Word devem ser abertos com software de escritório adicional
  instalado
  - Microsoft Word, [WPS Office](https://wps.com) para Windows
  - [iWork](https://en.wikipedia.org/wiki/IWork) para macOS
  - [WPS Office](https://wps.com), [LibreOffice](https://www.libreoffice.org/)
    para Linux
  - muitos sistemas de edição de documentos em linha como [Google
    Docs](https://docs.google.com/) também podem abrir e editar documentos Word
- Os documentos de Word podem ser apresentados de forma bastante diferente em diferentes
  softwares em diferentes plataformas
  - na realidade nem todos têm [Microsoft
    Word](https://products.office.com/en-us/word) instalado no computador
  - outro facto é que uma licença genuína para Microsoft Word pode custar
    centenas de dólares
- para tentar assegurar visualização e saída consistentes, os documentos
  Word são usados melhor com apenas algumas fontes comuns aos principais
  sistemas operativos, o que resulta em documentos Word com muito pouca liberdade na
  seleção de fontes.

Claro, diferentes gestores de RH em várias empresas podem ter
diferentes requisitos e recomendações. Recomendamos que exportes o currículo para
PDF como alternativa se o fizeres em Word

## Etiqueta de e-mail

À parte do sistema de recrutamento proprietário da empresa, o método
principal de enviar currículos para candidaturas a empregos deve ser o e-mail.
Algumas etiquetas básicas de e-mail são as seguintes.

- estabelece o昵称 de e-mail correto, não uses apelidos informais
  demasiado elaborados, isso dará ao empregador a impressão de ser pouco
  profissional
- usa um formato uniforme no cabeçalho do e-mail, mantém-no
  claro e conciso para transmitir a informação mais importante
  - formato recomendado: `Candidatar-Empresa Posição-Nome-[Universidade/Formação]-[Experiência Profissional]`
- o corpo do e-mail deve ser conciso, não é recomendado escrever
  uma longa carta de apresentação com muita avaliação subjetiva pessoal
  - melhor adicionar o título e uma saudação no início do texto
  - conteúdos no corpo
    - informação pessoal básica, como estado laboral (em serviço ou não),
      posição, formação e experiências profissionais, etc.
    - posição a que se candidata
    - enfoque da candidatura, se foi recomendada por outros
  - adicionar um assinante que contenha nome e informação de contacto
    (telefone/e-mail)
- anexa o arquivo do currículo nombrado com o mesmo formato do cabeçalho
