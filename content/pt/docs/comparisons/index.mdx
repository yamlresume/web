title: Comparações
---

O YAMLResume foi projetado para ser uma solução completa e opinativa para
currículos. Para ajudá-lo a entender como ele se encaixa no cenário mais amplo
de ferramentas e formatos de currículo, fornecemos comparações detalhadas com
as alternativas existentes.

Cada comparação aborda diferenças de funcionalidades, filosofia de design e
compromissos práticos para que você possa escolher a ferramenta certa para o
seu fluxo de trabalho.
