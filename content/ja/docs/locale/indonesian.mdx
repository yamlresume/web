インドネシア語（Bahasa Indonesia）は、世界最大の島嶼国家であるインドネシアの
[公用語](https://en.wikipedia.org/wiki/Indonesian_language)です。インドネシアの
多様な民族間で共通語として機能しており、2億7千万人以上が数百の言語を母語として
話しています。

## 言語学

### 文字セット

インドネシア語は英語と同じ[ラテン文字](https://en.wikipedia.org/wiki/Latin_alphabet)を使用します。追加文字なしの標準的な26文字のアルファベットに従います。

| 機能                 | 英語の文字セット           | インドネシア語の文字セット       |
| -------------------- | ------------------------ | ------------------------------ |
| **アルファベット**   | 26文字（`a-z`）          | 26文字（`a-z`）                |
| **ダイアクリティカルマーク** | なし              | なし                           |
| **特殊文字**         | 基本的なラテン文字に限定 | 基本的なラテン文字に限定       |

### 慣用句

#### 日付

インドネシア語の日付は、日-月-年の形式です。月の名前は头大文字で書きません。

| 英語      | インドネシア語 |
| --------- | ---------- |
| January   | Januari    |
| February  | Pebruari   |
| March     | Maret      |
| April     | April      |
| May       | Mei        |
| June      | Juni       |
| July      | Juli       |
| August    | Agustus    |
| September | September  |
| October   | Oktober    |
| November  | Nopember   |
| December  | Desember   |

#### 場所

インドネシアの住所は特定の形式に従います：

1.  **通りと番地**：通り名が最初に来て、その後に番地が続きます（例：「Jalan Sudirman No. 123」）。
2.  **町名（Kelurahan/Desa）**：最小行政単位。
3.  **地区（Kecamatan）**：次の行政単位。
4.  **都市（Kota/Kabupaten）**：市または郡。
5.  **州（Provinsi）**：最大の行政区画。
6.  **郵便番号**：5桁の郵便番号是最後に来ます。

#### 大文字の使用

インドネシア語は英語と同様の大小文字使用規則に従います：

- 文章の最初の単語は大文字で始めます。
- 固有名詞（人名、地名、組織名）は大文字で書きます。
- 曜日と月は头大文字で書きません。
- 言語名と国籍は头大文字で書きません。

### タイポグラフィ

#### 句読点

インドネシア語の句読点規則は英語と非常によく似ています：

- **引用符**：二重引用符 `"..."` を使用します（アメリカ式）。
- **アポストロフィ**：英語と同様に使用されます。
- **カンマ**：英語の規則に従います。
- **数字**：
  - **小数点**：カンマを使用します（例：`12,34`）。
  - **千の位区切り**：ピリオドを使用します（例：`1.234`）。
