---
title: Motor Markdown
---

El motor Markdown genera una versión en Markdown plano de su currículum. Esto es particularmente útil cuando necesita copiar y pegar el contenido de su currículum en sistemas de seguimiento de candidatos (ATS) u otros formularios que no admiten la carga de PDF.

## Configuración

Para usar el motor Markdown, especifique `engine: markdown` en su configuración de diseño.

```yaml
layouts:
  - engine: markdown
```

La salida de Markdown se generará junto con su PDF (si también tiene un diseño LaTeX configurado) o como un archivo independiente.
