---
title: Diseño
---

El núcleo de un currículum es el contenido, pero el diseño también es
importante. Muchos generadores de currículums basados en JSON/YAML solo se
enfocan en el contenido, dejando el diseño intacto, lo que deja un gran problema
para el usuario.

A diferencia de otros generadores de currículums, YAMLResume proporciona una
solución de diseño opinada de fábrica, que permite a los usuarios enfocarse en
el contenido del currículum mientras generan currículums de aspecto profesional
con diseño y composición perfectos a nivel de píxel.

Aquí hay un ejemplo de configuración de diseño en un archivo `resume.yml`:

```yml lineNumbers title="resume.yml"
---
content:
  # ...

locale:
  language: en

layouts:
  - engine: latex
    template: moderncv-banking
    page:
      showPageNumbers: true
      margins:
        top: 2.5cm
        left: 1.5cm
        right: 1.5cm
        bottom: 2.5cm
    typography:
      fontSize: 11pt
  - engine: markdown
```

## Locale

La opción `locale` te permite especificar el idioma para tu currículum. Esto
afectará el formato de fecha y otras configuraciones específicas del idioma.

```yml lineNumbers
locale:
  language: en
```

Echa un vistazo a [múltiples idiomas](/docs/content/multi-languages) para más
detalles.

## Márgenes

La opción `margins` te permite personalizar los márgenes de página. Ahora es parte del objeto `page` dentro de `layouts`.

```yml lineNumbers
layouts:
  - engine: latex
    page:
      margins:
        top: 2.5cm
        left: 1.5cm
        right: 1.5cm
        bottom: 2.5cm
```

## Página

La opción `page` te permite personalizar la configuración de página, como el
tamaño de papel y si mostrar números de página.

```yml lineNumbers
layouts:
  - engine: latex
    page:
      showPageNumbers: true
```

## Secciones

La opción `sections` te permite personalizar los títulos de sección y su orden.
Para más información, consulta la [documentación de
secciones](./layout/sections).

```yml lineNumbers
layouts:
  - engine: latex
    sections:
      aliases:
        work: "Professional Experience"
        education: "Education"
        skills: "Skills"
      order:
        - basics
        - work
        - education
```

## Plantilla

La opción `template` te permite elegir entre una variedad de plantillas. Cada
plantilla tiene un diseño y diseño únicos. Puedes encontrar una lista de
plantillas disponibles en la [documentación de plantillas](./layout/templates).

```yml lineNumbers
layouts:
  - engine: latex
    template: moderncv-banking
```

## Tipografía

La opción `typography` te permite personalizar los estilos de fuente usados en
tu currículum.

Actualmente, solo se soporta `fontSize`, y el motor LaTeX solo soporta
`10pt`, `11pt`, y `12pt`.

```yml lineNumbers
layouts:
  - engine: latex
    typography:
      fontSize: 11pt
```
