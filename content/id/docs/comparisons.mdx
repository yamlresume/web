## JSON Resume

[JSON Resume](https://jsonresume.org) adalah pionir dalam membangun standar terbuka
untuk format data terstruktur untuk resume. [Rilis resmi
pertamanya](https://jsonresume.org/blog/first-official-release) adalah pada 31 Juli
2014, lebih dari 10 tahun yang lalu.

Dari situs resminya:

```markdown
## Asal-usul dan Konseptualisasi

JSON Resume dikonseptualisasikan sebagai solusi terhadap lanskap format resume
yang terfragmentasi dan tidak konsisten. Resume tradisional, yang biasanya dibuat
dalam format seperti Microsoft Word atau PDF, sering mengalami masalah kompatibilitas
saat diurai oleh sistem yang berbeda. Mengenali masalah ini, Thomas Davis dan Roland Sharp
memulai JSON Resume sebagai proyek open-source untuk menciptakan format resume
universal yang dapat dibaca oleh mesin.

## Tonggak Pencapaian Pengembangan

Proyek mulai mendapat momentum pada awal tahun 2010-an, dengan pencapaian penting
seperti pembuatan skema JSON Resume, pengembangan alat CLI pertama, dan
pendirian registry JSON Resume. Tonggak pencapaian ini didorong oleh kontribusi
komunitas dan pengakuan yang semakin meningkat terhadap kebutuhan format resume
standar di industri teknologi.
```

JSON Resume telah melakukan pekerjaan yang luar biasa dalam membangun standar terbuka
untuk format data terstruktur untuk resume. Namun, ada beberapa fitur yang tidak
dimiliki YAMLResume.

YAMLResume sangat terinspirasi oleh [JSON Resume](https://jsonresume.org/), dan
lebih opinionated di beberapa aspek. Misalnya, JSON Resume tidak menjelaskan
cara mengkonversi data yang dapat dibaca mesin menjadi resume yang indah dan
dapat dibaca manusia.

Berikut adalah beberapa perbedaan penting:

| Fitur              | YAMLResume            | JSON Resume         |
| ------------------ | --------------------- | ------------------- |
| Format Data        | YAML                  | JSON                |
| Struktur Data      | Semua objek tingkat atas | Beberapa objek bersarang |
| Mesin Typesetting  | LaTeX                 | Tidak ada           |
| Spesifikasi Layout | Built-in              | Tidak ditentukan    |
| Dukungan Rich Text | Ya                    | Tidak               |

Kami akan menawarkan alat untuk mengimpor JSON Resume ke YAMLResume di masa depan.
