---
title: Mesi LaTeX
---

Mesin LaTeX adalah mesin inti dari YAMLResume, yang dirancang untuk menghasilkan
resume PDF berkualitas tinggi dan profesional menggunakan template LaTeX.

## Konfigurasi

Untuk menggunakan mesin LaTeX, tentukan `engine: latex` dalam konfigurasi tata
letak Anda.

```yaml
layouts:
  - engine: latex
    template: moderncv-banking
    # ... opsi lainnya
```

## Halaman

Opsi `page` memungkinkan Anda menyesuaikan pengaturan halaman, seperti ukuran
kertas, margin, dan apakah akan menampilkan nomor halaman.

```yml lineNumbers
layouts:
  - engine: latex
    page:
      showPageNumbers: true
      # a4 atau letter
      paperSize: a4
      margins:
        top: 2.5cm
        left: 1.5cm
        right: 1.5cm
        bottom: 2.5cm
```

## Bagian

Opsi `sections` memungkinkan Anda menyesuaikan judul bagian dan urutannya. Untuk
informasi lebih lanjut, lihat [dokumentasi bagian](./sections).

```yml lineNumbers
layouts:
  - engine: latex
    sections:
      aliases:
        work: "Pengalaman Profesional"
        education: "Pendidikan"
        skills: "Keterampilan"
      order:
        - basics
        - work
        - education
```

## Template

Opsi `template` memungkinkan Anda memilih dari berbagai template. Setiap template
memiliki desain dan tata letak yang unik. Anda dapat menemukan daftar template
yang tersedia di [dokumentasi template](./latex/templates).

```yml lineNumbers
layouts:
  - engine: latex
    template: moderncv-banking
```

## Tipografi

Opsi `typography` memungkinkan Anda menyesuaikan gaya font yang digunakan dalam
resume Anda.

### Ukuran Font

Opsi `fontSize` memungkinkan Anda mengatur ukuran font dasar untuk dokumen.
Mesin LaTeX hanya mendukung `10pt`, `11pt`, dan `12pt`.

### Keluarga Font

Opsi `fontFamily` memungkinkan Anda menentukan daftar prioritas nama keluarga
font. Ini bekerja serupa dengan properti CSS `font-family`.

### Spasi Baris

Opsi `lineSpacing` memungkinkan Anda menyesuaikan spasi baris (leading) dokumen.
YAMLResume menggunakan nilai semantik yang dipetakan ke perintah `\setstretch{}` LaTeX:

| Nilai    | Deskripsi                                   |
| -------- | ------------------------------------------ |
| `tight`  | Spasi kompak untuk muat lebih banyak konten |
| `snug`   | Sedikit lebih banyak ruang dari tight      |
| `normal` | Keseimbangan keterbacaan (default)         |
| `relaxed`| Lebih banyak ruang antar baris            |
| `loose`  | Tata letak sangat ruang                   |

```yaml
layouts:
  - engine: latex
    typography:
      fontSize: 11pt
      fontFamily: "Merriweather, serif"
      lineSpacing: relaxed
```

## Lanjutan

Opsi `advanced` memungkinkan Anda mengonfigurasi pengaturan lanjutan untuk mesin
LaTeX.

```yml lineNumbers
layouts:
  - engine: latex
    advanced:
      showIcons: true
      fontspec:
        numbers: Lining
```
