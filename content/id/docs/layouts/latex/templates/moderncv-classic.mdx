## Implementasi

Moderncv classic adalah template resume modern dan klasik berbasis
[xdanaux/moderncv](https://github.com/xdanaux/moderncv).

Di balik layar, kami menggunakan kode berikut untuk preambel LaTeX:

```latex lineNumbers
\documentclass[a4paper, serif, 11pt]{moderncv}

%% moderncv
% style and color
\moderncvstyle{classic}
\moderncvcolor{black}

% needed by moderncv for showing icons
\usepackage{fontawesome5}
```
