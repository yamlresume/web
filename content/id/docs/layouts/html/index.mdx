---
title: Mesin HTML
---

Mesin HTML menghasilkan versi resume HTML yang bersih dan siap untuk web. Ini
sangat berguna untuk menyematkan resume Anda di situs web, membagikannya secara
online, atau membuat portfolio berbasis web.

## Konfigurasi

Untuk menggunakan mesin HTML, tentukan `engine: html` dalam konfigurasi tata
letak Anda.

```yaml
layouts:
  - engine: html
    template: calm
    # ... opsi lainnya
```

## Bagian

Opsi `sections` memungkinkan Anda menyesuaikan judul bagian dan urutannya. Untuk
informasi lebih lanjut, lihat [dokumentasi bagian](./sections).

```yml lineNumbers
layouts:
  - engine: html
    sections:
      aliases:
        work: "Pengalaman Profesional"
        education: "Pendidikan"
        skills: "Keterampilan"
      order:
        - basics
        - work
        - education
```

## Template

Opsi `template` memungkinkan Anda memilih dari berbagai template. Setiap template
memiliki desain dan tata letak yang unik. Anda dapat menemukan daftar template
yang tersedia di [dokumentasi template](./html/templates).

```yml lineNumbers
layouts:
  - engine: html
    template: calm
```

## Tipografi

Opsi `typography` memungkinkan Anda menyesuaikan gaya font yang digunakan dalam
resume Anda.

### Ukuran Font

Opsi `fontSize` memungkinkan Anda mengatur ukuran font dasar untuk dokumen.
Mesin HTML mendukung ukuran font dari `14px` hingga `20px`.

### Keluarga Font

Opsi `fontFamily` memungkinkan Anda menentukan daftar prioritas nama keluarga
font dan/atau nama keluarga generik. Ini bekerja persis seperti properti CSS
`font-family`.

```yaml
layouts:
  - engine: html
    typography:
      fontSize: 16px
      fontFamily: "Merriweather, serif"
```

## Lanjutan

Opsi `advanced` memungkinkan Anda mengonfigurasi pengaturan lanjutan untuk mesin
HTML.

```yml lineNumbers
layouts:
  - engine: html
    advanced:
      showIcons: true
      # Judul kustom untuk dokumen HTML.
      title: "Judul Resume Kustom Saya"
      # Footer kustom untuk dokumen HTML.
      footer: "Dihasilkan oleh <a href=\"https://yamlresume.dev\">YAMLResume</a>"
      # Meta description untuk dokumen HTML.
      description: "Deskripsi singkat resume untuk SEO."
      # Meta keywords untuk dokumen HTML.
      keywords: "resume, cv, yamlresume"
```
