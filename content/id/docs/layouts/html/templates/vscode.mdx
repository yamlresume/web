## Implementasi

VS Code adalah tema gelap yang terinspirasi oleh skema warna "Dark+" Visual
Studio Code. Ini dirancang khusus untuk pengembang, insinyur perangkat lunak,
dan profesional teknis yang ingin resume mereka mencerminkan lingkungan
pengodean mereka.

Template mengadopsi estetika "Resume sebagai Kode", memperlakukan berbagai
bagian profil Anda sebagai elemen kode:

- **Nama**: Dihiasi sebagai `class` (mis., `class JohnDoe {`)
- **Headline**: Dirender sebagai komentar kode (mis., `// Full Stack Engineer`)
- **Bagian**: Dinyatakan sebagai pemanggilan fungsi (mis., `experience()`)
- **Keterampilan/Bahasa**: Gaya sebagai kunci properti objek (mis.,
  `JavaScript: Expert`)

Di balik layar, template menggunakan:

- **Tipografi**: Menggunakan stack font monospace (`Consolas`, `Monaco`,
  `Courier New`, `monospace`) untuk nuansa IDE yang autentik
- **Penyorotan Sintaks**: Menggunakan palet warna VS Code Dark+ resmi untuk
  kata kunci, string, class, dan fungsi
- **Indentasi Tata Letak**: Menampilkan panduan indentasi vertikal (border-left)
  untuk secara visual mewakili cakupan setiap bagian

## Fitur

- **Berpusat pada Pengembang**: Sempurna untuk menunjukkan keahlian teknis dan
  gairah dalam pengodean
- **Mode Gelap Default**: Tema gelap kontras tinggi yang mudah dilihat
- **Struktur Seperti Kode**: Metafora visual unik yang membuat resume teknis
  menonjol
- **Responsif**: Sepenuhnya diadaptasi untuk tampilan desktop dan seluler
  dengan semantik kode yang konsisten
- **Gaya Sintaks**: Menggunakan warna IDE yang familiar untuk membedakan
  peran, tanggal, dan deskripsi

## Pratinjau

Template VS Code memberikan tampilan yang khas dan modern yang langsung
terhubung dengan perekrut dan manajer perekrutan teknis. Ini mengubah riwayat
karier Anda menjadi "file sumber" yang bersih, dapat dibaca, dan familiar
secara visual.
