---
title: Bagian
---

YAMLResume menyediakan opsi fleksibel untuk mengkustomisasi bagian-bagian resume.
Anda dapat mengubah judul bagian mana pun dan mengatur ulang bagian-bagian agar
sesuai dengan profil profesional Anda. Kustomisasi ini ditangani dalam
konfigurasi `layouts.[].sections` di file `resume.yml` Anda.

```yml lineNumbers title="resume.yml"
---
content:
  # ...

layouts:
  - engine: latex
    sections:
      aliases:
        work: "Pengalaman Profesional"
        education: "Latar Belakang Pendidikan"
        skills: "Keterampilan Teknis"
      order:
        - work
        - education
        - awards
        - volunteer
```
