## Pendahuluan

Membuat resume mungkin tidak sulit, tetapi pasti adalah proses yang membosankan, panjang, dan melelahkan.

Dengan YAMLResume, Anda dapat membuat dan mengontrol versi resume Anda dengan YAML dan membuat PDF sempurna piksel dengan tata letak dan typesetting profesional dengan mudah.

YAMLResume adalah mesin inti dari [PPResume](https://ppresume.com), builder berbasis LaTeX. Ini telah membantu [ribuan orang](https://blog.ppresume.com/posts/2024-recap) di seluruh dunia membuat resume sempurna piksel dan profesional serta mendapatkan pekerjaan impian mereka. Di sini kami memutuskan untuk open source mesin ini, untuk memberikan orang hak dan kebebasan untuk mengatakan [tidak pada vendor lock-in](https://blog.ppresume.com/posts/no-vendor-lock-in)

![YAMLResume Baru dan Bangun Resume](/static/images/docs/yamlresume-new-and-build.webp)

## Memulai

### Pengguna Docker

Jika Anda pengguna docker, Anda beruntung, kami memiliki [image docker](https://hub.docker.com/r/yamlresume/yamlresume) untuk Anda, yang memiliki [yamlresume cli](https://www.npmjs.com/package/yamlresume) terinstal dengan semua dependensinya, seperti [XeTeX](https://www.tug.org/xetex/) dan beberapa [font yang direkomendasikan](/docs/installation#font).

Anda dapat membuat resume baru dengan perintah berikut:

```sh
docker run --rm -v $(pwd):/home/yamlresume yamlresume/yamlresume new my-resume.yml
```

Edit `my-resume.yml` sesuai kebutuhan lalu build resume dari yaml ke pdf:

```sh
docker run --rm -v $(pwd):/home/yamlresume yamlresume/yamlresume build my-resume.yml
```

### Pengguna macOS

Mari kita asumsikan Anda menggunakan macOS dengan [Homebrew](https://brew.sh/) sebagai manajer paket. Jika tidak, silakan lihat panduan [Instalasi](/docs/installation) untuk instruksi detail untuk sistem operasi Anda.

<Steps>

<Step>
#### Instal Node.js

Anda perlu [menginstal Node.js](https://nodejs.org/en/download/package-manager/all#macos) 20.x atau yang lebih baru.

```sh
brew install node
```

</Step>

<Step>
#### Instal Mesin Typesetting

YAMLResume membutuhkan [mesin typesetting](/docs/installation#typesetting-engine) untuk menghasilkan PDF. Saat ini mendukung dua mesin typesetting:

- [XeTeX](/docs/installation#xetex)
- [Tectonic](/docs/installation#tectonic)

<Tabs groupId="typesetting-engine" persist items={["XeTeX", "Tectonic"]}>
```sh tab="XeTeX"
brew install mactex
```

```sh tab="Tectonic"
brew install tectonic
```

</Tabs>
</Step>

<Step>
#### Instal YAMLResume

Menginstal `yamlresume` sangat mudah. Anda dapat menginstalnya menggunakan manajer paket Node.js favorit Anda:

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn', 'bun']}>
```sh tab="npm"
npm install -g yamlresume
```

```sh tab="pnpm"
pnpm add -g yamlresume
```

```sh tab="yarn"
yarn global add yamlresume
```

```sh tab="bun"
bun add -g yamlresume
```

</Tabs>

Anda juga dapat menginstal [`yamlresume` melalui
Homebrew](http://formulae.brew.sh/formula/yamlresume):

```sh
brew install yamlresume
```

Verifikasi `yamlresume` berhasil terinstal:

```console
$ yamlresume help
```

<Step>
#### [Opsional] Instal Font

##### Font Linux Libertine

Kami juga merekomendasikan menginstal [font Linux Libertine](/docs/installation#linux-libertine) untuk tampilan PDF terbaik, tetapi ini opsional.

```sh
brew install font-linux-libertine
```

##### Font Noto

Jika Anda perlu membuat resume CJK (China, Jepang, Korea), kami merekomendasikan untuk menginstal [font Noto](https://fonts.google.com/noto):

```sh
brew install font-noto-sans-cjk font-noto-serif-cjk
```

</Step>

<Step>
#### Buat Resume YAML Pertama Anda

OK, semuanya sudah siap, sekarang kita bisa membuat resume yaml dengan `yamlresume new`:

```console
$ yamlresume new my-resume.yml
✔ Created my-resume.yml successfully.
```

Edit `resume.yml` sesuai kebutuhan, lalu Anda dapat menghasilkan PDF dengan `yamlresume build`:

```console
$ yamlresume build my-resume.yml
◐ Generating resume PDF file with command: xelatex -halt-on-error my-resume.tex...
✔ Generated resumep PDF file successfully.
```

Anda juga dapat menggunakan `yamlresume dev` untuk build resume saat file berubah:

```console
$ yamlresume dev my-resume.yml
◐ Generating resume PDF file with command: xelatex -halt-on-error my-resume.tex...
✔ Generated resume PDF file successfully.
◐ Watching file changes: my-resume.yml...
```

</Step>
</Steps>
