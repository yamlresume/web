## Format File

Format file umum untuk resume meliputi:

- PDF
- Microsoft Word (doc, docx)
- Plain Text
- HTML

Di sini kami sangat merekomendasikan PDF:

- PDF memiliki kualitas output lintas platform yang sangat baik, memastikan
  pencetakan dan tampilan yang hampir 100% konsisten
- PDF dapat menyematkan font, memberikan kebebasan besar dalam pemilihan font
- pembaca PDF default tersedia di hampir semua platform
  - Windows 8 ke atas memiliki pembaca PDF bawaan
  - macOS memiliki
    [Preview](<https://en.wikipedia.org/wiki/Preview_(Mac_OS)>), iOS
    memiliki [iBooks](http://www.apple.com/ibooks/)
  - banyak browser (seperti Chrome) memiliki
    [pdf.js](http://mozilla.github.io/pdf.js/) bawaan, yang juga dapat
    membuka PDF secara default

Lebih mudah untuk mempelajari MS Word, tetapi mengirim resume dengan format
Word memiliki kekurangan yang besar:

- Dokumen Word harus dibuka dengan perangkat lunak kantor tambahan yang
  terpasang
  - [Microsoft Word](https://products.office.com/en-us/word), [WPS
    Office](https://wps.com) untuk Windows
  - [iWork](https://en.wikipedia.org/wiki/IWork) untuk macOS
  - [WPS Office](https://wps.com),
    [LibreOffice](https://www.libreoffice.org/) untuk Linux
  - banyak sistem pengeditan dokumen online seperti [Google
    Docs](https://docs.google.com/) juga dapat membuka dan mengedit dokumen
    Word
- Dokumen Word dapat ditampilkan sangat berbeda di perangkat lunak berbeda
  pada platform berbeda
  - pada kenyataannya tidak semua orang telah menginstal [Microsoft
    Word](https://products.office.com/en-us/word) di komputer mereka
  - fakta lain adalah bahwa lisensi asli untuk Microsoft Word dapat
    berharga ratusan dolar
- untuk mencoba memastikan tampilan dan output yang konsisten, dokumen Word
  paling baik digunakan hanya dengan beberapa font yang umum ada di sistem
  operasi utama, yang menghasilkan dokumen Word dengan kebebasan yang sangat
  sedikit dalam pemilihan font

Tentu saja, manajer HR yang berbeda di berbagai perusahaan dapat memiliki
persyaratan dan preferensi berbeda. Kami merekomendasikan Anda mengekspor
resume ke PDF sebagai alternatif jika membuatnya dengan Word

## Etiket Email

Selain sistem rekrutmen proprietary perusahaan, metode utama pengiriman resume
untuk melamar pekerjaan adalah email. Beberapa etiket email dasar adalah
sebagai berikut.

- gunakan nickname email yang benar, jangan gunakan nickname yang terlalu
  unik dan tidak formal, itu akan memberikan kesan kepada pemberi kerja bahwa
  Anda sangat tidak profesional
- gunakan format seragam di header email, jelaskan dan singkat untuk
  menyampaikan informasi paling penting
  - format yang disarankan: `Lamar-Posisi-Perusahaan-Nama-[Universitas/Pendalaman]-[Pengalaman Kerja]`
- tubuh email harus singkat, tidak disarankan untuk menulis surat lamaran
  panjang dengan banyak evaluasi subjektif pribadi
  - yang terbaik adalah menambahkan judul dan salam di awal teks
  - isi dalam tubuh
    - informasi pribadi dasar, seperti status kerja (bekerja atau tidak),
      posisi, pendidikan dan pengalaman kerja, dll.
    - posisi yang dilamar
    - cara melamar, apakah direkomendasikan oleh orang lain
  - tambahkan penanda tangan yang berisi nama dan informasi kontak
    (telepon/email)
- lampirkan file resume dengan nama format yang sama dengan header
