Perancis adalah bahasa [Romawi](https://en.wikipedia.org/wiki/Romance_languages)
dari keluarga Indo-Eropa. Meskipun berbagi aksara Latin dengan bahasa Inggris, ia
memiliki karakteristik berbeda dalam tipografi, tanda baca, dan tata bahasa yang
memerlukan pertimbangan khusus untuk YAMLResume.

## Linguistik

### Set Karakter

Perancis menggunakan 26 huruf dasar [Aksara
Latin](https://en.wikipedia.org/wiki/Latin_alphabet), dengan empat diakritik
yang muncul pada vokal (aksen sirkumfleks, diaeresis, aksen grave, aksen akut)
dan cedilla muncul dalam "ç". Ada juga dua ligatur, "œ" dan "æ".

| Fitur                | Set Karakter Bahasa Inggris             | Set Karakter Bahasa Perancis                                                                                     |
| ---------------------- | --------------------------------- | -------------------------------------------------------------------------------------------------------- |
| **Alphabet**           | 26 huruf (`a-z`)                | 26 huruf ditambah huruf dengan diakritik dan ligatur  |
| **Diakritik**         | Tidak ada                              | Akut (`é`), grave (`à`, `è`, `ù`), sirkumfleks (`â`, `ê`, `î`, `ô`, `û`), diaeresis (`ë`, `ï`, `ü`, `ÿ`), cedilla (`ç`) |
| **Karakter Khusus** | Terbatas pada karakter Latin dasar | Menyertakan ligatur `œ` (misalnya "cœur") dan `æ` (misalnya "ex æquo") |

### Idiom

#### Tanggal

Hari dalam seminggu dan bulan tidak dikapitalisasi dalam bahasa Perancis (tidak seperti bahasa Inggris).
Misalnya, "Sep 2021" dalam bahasa Inggris diterjemahkan menjadi "sept. 2021" dalam bahasa Perancis.

YAMLResume hanya menggunakan bulan, oleh karena itu tabel terjemahan:

| --------------- | ---------------- |
| January (Jan)   | janvier (janv.)  |
| February (Feb)  | février (févr.)  |
| March (Mar)     | mars (mars)      |
| April (Apr)     | avril (avr.)     |
| May (May)       | mai (mai)        |
| June (Jun)      | juin (juin)      |
| July (Jul)      | juillet (juil.)  |
| August (Aug)    | août (août)      |
| September (Sep) | septembre (sept.)|
| October (Oct)   | octobre (oct.)   |
| November (Nov)  | novembre (nov.)  |
| December (Dec)  | décembre (déc.)  |

#### Kapitalisasi

Perancis umumnya mengikuti huruf kalimat untuk judul dan heading, tetapi YAMLResume
menyediakan istilah yang dikapitalisasi untuk konsistensi dengan standar resume.

Satu perbedaan notable adalah bahwa bahasa dan nama kebangsaan yang digunakan sebagai kata sifat
tidak dikapitalisasi dalam bahasa Perancis standar (misalnya "je parle français"), tetapi
mereka dikapitalisasi ketika merujuk pada kata benda (bahasa itu sendiri) dalam daftar dalam
YAMLResume untuk konsistensi.

### Typesetting

#### Tanda Baca

Perancis memiliki aturan spesifik untuk spasi di sekitar tanda baca:

- **Tanda baca tinggi**: titik dua (`:`), titik koma (`;`), tanda seru
  (`!`), dan tanda tanya (`?`) memerlukan spasi non-breaking sebelum它们.
- **Tanda kutip**: Perancis menggunakan guillemets (`«` dan `»`) dengan spasi
  non-breaking di dalamnya (misalnya `« Bonjour »`).
- **Pemisah desimal**: Perancis menggunakan koma sebagai pemisah desimal (misalnya,
  3,14).
- **Pemisah ribuan**: Perancis menggunakan spasi non-breaking sebagai pemisah ribuan
  (misalnya 1 000).

YAMLResume menangani spasi titik dua secara spesifik untuk tata letak Perancis (misalnya ` :`).
