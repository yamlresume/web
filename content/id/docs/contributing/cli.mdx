title: CLI
---

YAMLResume mengadopsi struktur mono-repo, yang berarti semua paket berada dalam
satu repository. Berkat [pnpm
workspace](https://pnpm.io/workspaces), pendekatan ini menyederhanakan
manajemen dependensi dan versioning di seluruh paket.

Pada saat penulisan, repo YAMLResume memiliki dua paket:

- [yamlresume/cli](https://github.com/yamlresume/yamlresume/tree/main/packages/cli):
  antarmuka CLI untuk YAMLResume.
- [yamlresume/core](https://github.com/yamlresume/yamlresume/tree/main/packages/core):
  berisi mesin inti untuk YAMLResume. Ini menangani penguraian, validasi, dan
  rendering resume berdasarkan [skema resmi](/docs/compiler/schema).

## Build yamlresume/core

`yamlresume/cli` bergantung pada `yamlresume/core` untuk berfungsi dengan baik.
Jadi ketika Anda ingin menguji `yamlresume/cli`, Anda perlu memastikan bahwa
`yamlresume/core` juga di-build dan tersedia.

Anda dapat melakukan ini dengan perintah berikut:

```bash
# build untuk tujuan dev
pnpm core build

# build untuk produksi, yang akan minifikasi output dan menghasilkan file
# definisi TypeScript `.d.ts`
pnpm core build:prod

# pantau perubahan file dan build secara otomatis
pnpm core build:watch
```

## Build yamlresume/cli

Setelah `yamlresume/core` di-build, Anda dapat menjalankan perintah CLI dengan dua
cara:

1. build `yamlresume/cli` secara lokal dan jalankan node.js dengan `dist/cli.js`:

```bash
pnpm cli build
node packages/cli/dist/cli.js <command> [options]
```

2. jalankan `yamlresume/cli` langsung dengan bantuan [tsx](https://tsx.is/):

```bash
pnpm cli dev <command> [options]

# perintah ini akan diterjemahkan ke
# tsx src/cli.ts
```
