---
title: Ekosistem
---

YAMLResume menyediakan serangkaian alat untuk membantu Anda membuat, mengonversi,
dan mengelola resume Anda dengan lebih efisien. Berikut adalah beberapa utilitas
utama yang tersedia:

## create-yamlresume

[create-yamlresume](./ecosystem/create-yamlresume) memudahkan Anda memulai proyek
YAMLResume baru dengan perintah satu baris. Alat ini akan membuat struktur direktori
proyek, menginstal dependensi yang diperlukan, dan menghasilkan file contoh resume
sehingga Anda dapat langsung memulai.

- Merancang proyek baru dengan `npx create-yamlresume my-resume` atau perintah
  serupa untuk `npm`, `yarn`, atau `pnpm`.
- Menyertakan struktur proyek yang siap digunakan, skrip untuk membangun dan
  memvalidasi resume, serta contoh file resume YAML.

## json2yamlresume

[json2yamlresume](./ecosystem/json2yamlresume) adalah alat CLI untuk mengonversi
file [JSON Resume](https://jsonresume.org/) ke format YAMLResume.

- Mengonversi file JSON Resume ke YAMLResume, mengubah struktur dan kolom sesuai
  kebutuhan.
- Memastikan data Anda divalidasi terhadap skema YAMLResume.

Alat-alat ini membantu Anda bermigrasi dari format lain, mengotomatiskan penyiapan
proyek, dan menyederhanakan alur kerja Anda dengan YAMLResume.
