Kami dengan sukacita mengumumkan rilis
[YAMLResume v0.10](https://github.com/yamlresume/yamlresume/releases/tag/v0.10.0).
Rilis ini berfokus pada perluasan ekosistem layout kami dengan tema favorit
developer, memperluas dukungan internasional kami, dan menambahkan kontrol
terperinci atas pemformatan dokumen.

## Template HTML Baru: VS Code

Untuk developer, editor kode Anda adalah rumah Anda. Sekarang curriculum vitae
Anda bisa terasa seringkas itu. Kami telah memperkenalkan template [VS
Code](/docs/layouts/html/templates/vscode) baru untuk mesin HTML, menampilkan
tema gelap yang terinspirasi oleh skema warna "Dark+" dari Visual Studio
Code.

![YAMLResume VS Code Template](/static/images/docs/layouts/html/templates/vscode.webp)

Ini menampilkan estetika syntax-highlighting—memperlakukan nama Anda seperti
class, keterampilan Anda seperti objek, dan pengalaman Anda seperti fungsi.
Untuk menggunakannya, cukup perbarui konfigurasi layout Anda:

```yml lineNumbers
layouts:
  - engine: html
    template: vscode  #[!code highlight]
```

Template ini sempurna untuk insinyur perangkat lunak, profesional devops, dan
siapa saja yang ingin langsung menunjukkan akar teknis mereka.

## Dukungan Bahasa Belanda

YAMLResume terus berkembang secara global. Berkat kontribusi dari
[MattiaPun](https://github.com/MattiaPun), kami sekarang mendukung [Belanda
(nl)](/docs/locale/dutch)!

Ini membawa jumlah bahasa yang kami dukung menjadi enam:

- [Inggris (en)](/docs/locale/english)
- [Tionghoa (zh)](/docs/locale/chinese)
- [Spanyol (es)](/docs/locale/spanish)
- [Prancis (fr)](/docs/locale/french)
- [Norwegia (no)](/docs/locale/norwegian)
- [Belanda (nl)](/docs/locale/dutch)

Untuk menghasilkan curriculum vitae Anda dalam bahasa Belanda, atur lokal di
`resume.yml` Anda:

```yml lineNumbers
locale:
  language: nl
```

Ini akan secara otomatis menerjemahkan header bagian (misalnya,
"Werkervaring", "Opleidingen"), format tanggal, dan nama negara.

## Dukungan Ukuran Kertas LaTeX

Kami telah menambahkan kontrol eksplisit atas ukuran kertas untuk mesin
LaTeX/PDF. Meskipun A4 adalah standar di banyak bagian dunia, US Letter
penting untuk Amerika Utara.

Anda sekarang dapat menentukan ukuran kertas yang Anda inginkan langsung dalam
konfigurasi:

```yml lineNumbers
layouts:
  - engine: latex
    page:
      # Options: 'a4' (default) or 'letter'
      paperSize: letter   #[!code highlight]
```

Ini memastikan curriculum vitae PDF Anda cetakan sempurna terlepas dari
region Anda. Terima kasih khusus kepada [djryanj](https://github.com/djryanj)
untuk mendorong fitur ini!

## Toggle Ikon

Terkadang kurang adalah lebih. Meskipun ikon menambah daya tarik visual,
konteks formal tertentu atau yang berfokus pada ATS mungkin memerlukan
presentasi teks saja yang lebih ketat.

Kami telah memperkenalkan opsi `showIcons` di pengaturan `advanced` untuk mesin
HTML dan LaTeX. Ini memberi Anda fleksibilitas untuk mengaktifkan atau
menonaktifkan ikon untuk detail kontak dan profil sosial.

```yml lineNumbers
layouts:
  - engine: html
    advanced:
      showIcons: false  #[!code highlight]
  - engine: latex
    advanced:
      showIcons: false  #[!code highlight]
```
