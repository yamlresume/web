---
title: Mise en Page
---

Le cœur d'un CV est le contenu, mais la mise en page est aussi importante. Beaucoup de
générateurs de CV basés sur JSON/YAML se concentrent seulement sur le contenu, laissant la
mise en page intacte, ce qui laisse un gros problème pour l'utilisateur.

Contrairement aux autres générateurs de CV, YAMLResume fournit une solution de mise en page
opiniâtre prête à l'emploi, qui permet aux utilisateurs de se concentrer sur le contenu du
CV tout en générant des CV d'apparence professionnelle avec une mise en page et une
composition parfaites au pixel près.

Voici un exemple de configuration de mise en page dans un fichier `resume.yml` :

```yml lineNumbers title="resume.yml"
---
content:
  # ...

layout:
  locale:
    language: en
  margins:
    top: 2.5cm
    left: 1.5cm
    right: 1.5cm
    bottom: 2.5cm
  page:
    showPageNumbers: true
  template: moderncv-banking
  typography:
    fontSize: 11pt
```

## Locale

L'option `locale` vous permet de spécifier la langue pour votre CV. Cela
affectera le format de date, et d'autres paramètres spécifiques à la locale.

```yml lineNumbers
layout:
  locale:
    language: en
```

Consultez [multi-langues](/fr/docs/content/multi-languages) pour plus
de détails.

## Marges

L'option `margins` vous permet de personnaliser les marges de page.

```yml lineNumbers
layout:
  margins:
    top: 2.5cm
    left: 1.5cm
    right: 1.5cm
    bottom: 2.5cm
```

## Page

L'option `page` vous permet de personnaliser les paramètres de page, comme la taille du papier
et s'il faut afficher les numéros de page.

```yml lineNumbers
layout:
  page:
    showPageNumbers: true
```

## Sections

L'option `sections` vous permet de personnaliser les titres de section et leur
ordre. Pour plus d'informations, voir la [documentation des sections](./layout/sections).

```yml lineNumbers
layout:
  sections:
    aliases:
      work: "Expérience Professionnelle"
      education: "Éducation"
      skills: "Compétences"
    order:
      - basics
      - work
      - education
```

## Modèle

L'option `template` vous permet de choisir parmi une variété de modèles. Chaque
modèle a un design et une mise en page uniques. Vous pouvez trouver une liste des
modèles disponibles dans la [documentation des modèles](./layout/templates).

```yml lineNumbers
layout:
  template: moderncv-banking
```

## Typographie

L'option `typography` vous permet de personnaliser les styles de police utilisés dans votre CV.

Actuellement, seul `fontSize` est supporté, et le moteur LaTeX supporte seulement
`10pt`, `11pt`, et `12pt`.

```yml lineNumbers
layout:
  typography:
    fontSize: 11pt
```
