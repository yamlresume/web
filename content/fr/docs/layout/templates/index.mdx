---
title: Modèles
---

YAMLResume fournit plusieurs modèles prêts à l'emploi, vous pouvez lister tous les modèles
avec la commande `yamlresume templates list`.

```console
$ yamlresume templates list

| `layout.template` | Nom du Modèle    | Description                          |
| ----------------- | ---------------- | ------------------------------------ |
| moderncv-banking  | ModernCV Banking | Modèle ModernCV avec style banking   |
| moderncv-casual   | ModernCV Casual  | Modèle ModernCV avec style casual    |
| moderncv-classic  | ModernCV Classic | Modèle ModernCV avec style classic   |
```

Par défaut, YAMLResume utilise le modèle [moderncv-banking](./templates/moderncv-banking). Vous
pouvez changer le modèle en définissant `layout.template` dans votre CV YAML.

```yml lineNumbers title="resume.yml"
content:
  # ...

layout:
  # ...
  template: moderncv-classic # [!code highlight]
  # ...
```
