L'indonésien (Bahasa Indonesia) est la
[langue officielle](https://en.wikipedia.org/wiki/Indonesian_language) de
l'Indonésie, la plus grande nation archipelagique du monde. Elle sert de lingua
franca entre la population diversifiée de l'Indonésie, avec plus de 270 millions
de personnes parlant des centaines de langues natives.

## Linguistique

### Jeu de Caractères

L'indonésien utilise l'[alphabet latin](https://en.wikipedia.org/wiki/Latin_alphabet),
le même système d'écriture que l'anglais. Il suit l'alphabet standard de 26
lettres sans caractères supplémentaires.

| Fonctionnalité          | Jeu de caractères anglais         | Jeu de caractères indonésien        |
| ---------------------- | -------------------------------- | ---------------------------------- |
| **Alphabet**           | 26 lettres (`a-z`)              | 26 lettres (`a-z`)                 |
| **Diacritiques**       | Aucun                            | Aucun                              |
| **Caractères Spéciaux** | Limité aux caractères latins de base | Limité aux caractères latins de base |

### Idiomes

#### Date

Les dates en indonésien suivent le format jour-mois-année. Les noms des mois ne
sont pas mis en majuscule.

| --------- | ---------- |
| January   | Januari    |
| February  | Pebruari   |
| March     | Maret      |
| April     | April      |
| May       | Mei        |
| June      | Juni       |
| July      | Juli       |
| August    | Agustus    |
| September | September  |
| October   | Oktober    |
| November  | Nopember   |
| December  | Desember   |

#### Emplacement

Les adresses indonésiennes suivent un format spécifique :

1.  **Rue et Numéro** : Le nom de la rue vient en premier, suivi du
    numéro de maison (exemple : "Jalan Sudirman No. 123").
2.  **Quartier (Kelurahan/Desa)** : La plus petite zone administrative.
3.  **Arrondissement (Kecamatan)** : Le niveau administratif suivant.
4.  **Ville (Kota/Kabupaten)** : La ville ou la régence.
5.  **Province (Provinsi)** : La plus grande division administrative.
6.  **Code Postal** : Le code postal à 5 chiffres vient à la fin.

#### Majuscules

L'indonésien suit des règles de majuscules similaires à l'anglais :

- Mettez une majuscule au premier mot d'une phrase.
- Mettez une majuscule aux noms propres (noms de personnes, lieux, organisations).
- Les jours et les mois ne sont pas mis en majuscule.
- Les noms de langues et les nationalités ne sont pas mis en majuscule.

### Typographie

#### Ponctuation

Les règles de ponctuation indonésiennes sont très similaires à l'anglais :

- **Guillemets** : Utilise des guillemets doubles `"..."` (style américain).
- **Apostrophe** : Utilisée de manière similaire à l'anglais.
- **Virgule** : Suit les règles de l'anglais.
- **Nombres** :
  - **Séparateur décimal** : Utilise une virgule (exemple : `12,34`).
  - **Séparateur de milliers** : Utilise un point (exemple : `1.234`).
