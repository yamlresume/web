YAMLResume est un outil Node.js pour créer des CV en PDF à partir de fichiers YAML.

## Docker (Recommandé)

La méthode la plus simple est d'utiliser Docker :

```sh
docker run --rm -v $(pwd):/home/yamlresume yamlresume/yamlresume new my-resume.yml
```

```sh
docker run --rm -v $(pwd):/home/yamlresume yamlresume/yamlresume build my-resume.yml
```

## Installation Manuelle

### 1. Installer Node.js

Vous avez besoin de Node.js 20.x ou plus récent.

**macOS:**
```sh
brew install node
```

**Ubuntu/Debian:**
```sh
sudo apt install nodejs npm
```

### 2. Installer YAMLResume

```sh
npm install -g yamlresume
```

### 3. Installer un Moteur LaTeX

**Option A: XeTeX (recommandé)**

macOS:
```sh
brew install --cask mactex
```

Ubuntu/Debian:
```sh
sudo apt install texlive-xetex texlive-latex-extra
```

**Option B: Tectonic**

macOS:
```sh
brew install tectonic
```

Ubuntu/Debian:
```sh
sudo apt install tectonic
```

## Vérification

Testez votre installation :

```console
$ yamlresume new test.yml
✔ Created test.yml successfully.

$ yamlresume build test.yml
✔ Generated resume PDF file successfully.
```

Si tout fonctionne, vous avez maintenant un fichier `test.pdf`.
